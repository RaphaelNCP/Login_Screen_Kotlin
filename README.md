# Simulador de tela de login

Projeto realizado para testar minhas recem adiquiridas habilidades em kotlin. O projeot baseia-se em uma simulação de tela de login, contendo pagina de login, cadastro, recuperar senha, conclusão de cadastro e conclusão de login
